# Kyma Control Plane - Subscription Cleanup Job

## Overview

Subscription Cleanup Job is a subcomponent of [Kyma Control Plane](https://github.com/kyma-project/control-plane) for cleanup of managed subscriptions from [HAP](https://github.com/kyma-project/kyma-environment-broker/blob/main/docs/contributor/03-10-hyperscaler-account-pool.md).

For more information on KCP and its components, read the [KCP documentation](https://github.com/kyma-project/control-plane/tree/main/docs) and the [KEB documentation](https://github.com/kyma-project/kyma-environment-broker/blob/main/docs).
